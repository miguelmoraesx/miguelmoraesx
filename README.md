# 💻 Projetos

## 🔐 PIN Brute Force — Paralelismo em Sistemas Operacionais

Implementação de um ataque de **força bruta contra PIN de 6 dígitos**, com versões sequencial e paralela.

A otimização substituiu **mutex por operações atômicas**, reduzindo o tempo de execução de:

**23ms → 5,6ms com 4 threads**

Tecnologias utilizadas:

C • pthreads • Linux • Programação Paralela

---

## 📡 Android Network Scanner

Aplicação Android desenvolvida em **Java** capaz de identificar **portas abertas em um endereço IP**.

Funcionalidades:

* Entrada de IP alvo
* Definição de intervalo de portas
* Detecção de portas abertas via **sockets**
* Interface Android baseada em **XML**
